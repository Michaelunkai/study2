Step-by-Step Guide to Get a Notion API Key:
1. Sign Up/Log In:
   - Go to https://www.notion.so and create an account or log in.

2. Create an Integration:
   - Visit the Notion Developers Portal: https://developers.notion.com/
   - Click "Create new integration."

3. Fill in Details:
   - Name your integration and select the workspace it will connect to.

4. Copy the API Key:
   - After creating the integration, the API key (secret token) will be shown. Copy it.

5. Grant Integration Access:
   - Share specific pages or databases with the integration.
