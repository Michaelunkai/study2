Step-by-Step Guide to Get a Google Sheets API Key:
1. Set Up Google Cloud Account:
   - Go to https://console.cloud.google.com/ and create a free account.

2. Create a New Project:
   - Navigate to the project selection page and create a new project.

3. Enable the API:
   - Search for "Google Sheets API" in the library and enable it.

4. Generate Credentials:
   - Go to the "Credentials" tab.
   - Select "Create credentials" > "API key."

5. Restrict the API Key:
   - Optionally, limit the key to specific IPs or APIs for security.
